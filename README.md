




### Features

- Boostrapping Checker

- Friendly UI
- Compatible Windows 10/11
## Tech Stack

**PC Checking Tool:** Assembly, C#, PowerShell

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/playboifusi/Celine/
    ```

2. Navigate to the project directory:

    ```bash
    cd Celine
    ```

3. Run the project:

    ```bash
    dotnet run
    ```
